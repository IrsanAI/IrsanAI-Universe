## Kern-Tokens (NeuroToken Vocabulary)
- **Flux**: State Transition / Wandel / Change / Flow  
- **Anchor**: Stabile Referenz / Baseline / Fixed Point  
- **Mirror**: Selbst-Reflexion / Meta-Reasoning  
- **Pulse**: Prioritäts-Signal / Escalation  
- **Weave**: Synthese / Integration / Merge  
- **Relay**: Übergabe / Handoff  
- **Horizon**: Zukunfts-Grenze / Projection  
- **Resonance**: Alignment / Resonanz-Check  
- **Folding**: Kompression / Fold / Compress  
- **Consensus**: Multi-Model-Übereinstimmung / Trinity  

## Zweck
Token-Effizienz in Multi-Agent-Systemen (PDP v3.0 / LRP v1.3).  
Natürlich lesbar für Menschen, hochdicht für Agents.
