# IrsanAI Neural Token Folding (NTF)
## An Immersive Compression Standard for Agentic LLM Communication

**Status:** Conceptual Framework / Draft v1.0  
**Datum:** Februar 2026  
**Multi-Model Consensus:** Grok (xAI) · Gemini (Google) · Claude (Anthropic)

---

## Abstract
NTF schlägt eine immersive, semantisch gefaltete Kompression vor, die den ursprünglichen Inhalt auf ca. 5–20 % der Original-Tokenanzahl reduziert – bei nahezu verlustfreier semantischer Treue. Es nutzt das "NeuroToken Vocabulary", um komplexe Zustände in hochdichte Symbole zu falten.

## Architektur (The Core Folding)



1. **Encoder:** Analyse des Raw Context (Tokens → Embeddings).
2. **Latent Space Projection:** Query-aware Verdichtung basierend auf der Agenten-Intention.
3. **Folding:** Quantisierung in NeuroTokens (Flux, Anchor, Pulse etc.).

## NeuroToken Vocabulary
| Token     | Semantische Bedeutung              | Verwendung                          |
|-----------|------------------------------------|-------------------------------------|
| `Flux`    | State-Transition                   | Phasenwechsel signalisieren         |
| `Anchor`  | Stabile Referenz                   | Gemeinsame Annahmen fixieren        |
| `Mirror`  | Self-Evaluation-Loop               | Meta-Reasoning-Pass                 |
| `Pulse`   | Prioritäts-Eskalation              | Hochrelevante Signale               |
| `Weave`   | Cross-Domain-Synthese              | Integration mehrerer Quellen        |
| `Relay`   | Kontrollierte Übergabe             | Abstraktion weiterleiten            |

## Hypothetische Zielwerte
- **Kompressionsrate:** 15–25×
- **Semantische Treue:** ≥ 0.98 (near-lossless)
- **Rekonstruierbarkeit:** Vollständig via Unfolding Decoder.
